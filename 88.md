## 二、系统架构概述

### 2.1 系统组成
```
A3_sz/
├── api_server.py          # FastAPI后端服务
├── app_gradio.py          # Gradio前端界面
├── component/             # 核心组件模块
│   ├── agent.py          # 智能体核心入口
│   ├── config.py         # 全局配置管理
│   ├── files_parser.py   # 文件处理模块
│   ├── llms.py           # LLM模型管理
│   ├── logger.py         # 日志管理
│   ├── memory_sqlite.py  # 会话记忆持久化
│   ├── models.py         # 数据模型定义
│   ├── rag_course.py     # 课程RAG检索
│   ├── recommender.py    # 问题推荐系统
│   ├── session_manager.py # 会话管理
│   ├── sql_agent.py      # SQL查询代理
│   ├── tavily_search.py  # 网络检索工具
│   └── tools.py          # 工具注册与执行
├── assets/               # 静态资源
│   ├── script.js
│   ├── style.css
│   └── ...
├── data/                 # 数据存储目录
│   ├── chroma_course/   # 向量数据库
│   ├── course_doc/      # 课程文档
│   ├── general_files/   # 通用文件
│   ├── log_sql/         # 日志文件
│   └── temp/            # 临时文件
└── requirements.txt      # 项目依赖
```

### 2.2 核心功能模块

#### 2.2.1 前端界面（app_gradio.py）
- 课程问答核心界面
- 意图识别与路由
- 课程知识库管理
- 多用户/多会话隔离
- 工具调用统计
- 健康检查

#### 2.2.2 后端服务（api_server.py）
- 课程问答核心接口
- 意图识别与路由接口
- 课程知识库管理接口
- 多用户/多会话隔离接口
- 工具调用统计接口
- 健康检查接口

#### 2.2.3 核心组件
1. **智能体模块（agent.py）**
   - 意图识别与分类
   - 多模态交互支持
   - 会话持久化管理
   - 标准化响应处理

2. **RAG检索模块（rag_course.py）**
   - 文档加载与处理
   - 向量库初始化与管理
   - 语义检索与问答

3. **文件处理模块（files_parser.py）**
   - 多格式文件解析（PDF/DOCX/PPTX/TXT/HTML/IPYNB）
   - MinIO对象存储集成
   - 文件验证与图标系统

4. **会话管理模块（memory_sqlite.py）**
   - SQLite持久化存储
   - 多会话隔离
   - 聊天历史管理

5. **工具系统（tools.py）**
   - 工具注册与管理
   - 工具执行监控
   - 统一调用接口
